Features

Everything you need to build and serve dynamic web applications

Async I/O

Built on Boost.Asio with a thread pool for fully non-blocking request handling. Handles thousands of concurrent connections efficiently.

Lua Scripting

Write your request handlers in Lua with full access to request/response objects, SQLite databases, file I/O, and HTML template rendering.

HTTPS / SSL

Native TLS support via OpenSSL. Simply provide your certificate and key files in the configuration to enable secure connections.

SQLite Integration

Built-in SQLite bindings accessible from Lua. Create tables, run queries, and manage data without any external database server.

Template Engine

Embedded template engine with {{ var }} for escaped output, {! raw !} for HTML, and {% lua %} for logic blocks.

JSON Configuration

Configure port, SSL, Lua scripts, web root, logging, and log rotation through a single server_config.json file.

Logging & Rotation

Colored console output and file logging with automatic size-based rotation. Tracks request method, path, status, response size, and duration.

File Uploads

Multipart form parsing with binary-safe handling. Upload images, documents, ZIP files and more with original filename preservation.

CAPTCHA Generation

Server-side BMP image CAPTCHA generation in C++ with noise, random lines, and per-character color variation � no external libraries needed.
