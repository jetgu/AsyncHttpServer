About This Project

This is a demonstration of serving HTML files via Lua scripts in a C++ async HTTP server.

Technology Stack

  • C++14 - Core server implementation
  • Boost.Asio - Async I/O operations
  • OpenSSL - TLS/HTTPS support
  • Lua 5.4 - Scripting engine

How It Works

The Lua script reads HTML files from disk and serves them with the correct MIME type.
